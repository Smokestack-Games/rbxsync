Get Started

Installation

Up and running in minutes with a few simple commands.

1

Build from Source

Clone the repository and build with Cargo.

# Clone and build
git clone https://github.com/devmarissa/rbxsync
cd rbxsync
cargo build --release
2

Install Studio Plugin

Build and install the Roblox Studio plugin.

./target/release/rbxsync build-plugin --install
3

Initialize Project

Create a new project and start the server.

rbxsync init --name MyGame
rbxsync serve
4

Connect Studio

Open Roblox Studio, set the project path in the RbxSync widget, and click Connect. You're ready to sync!
